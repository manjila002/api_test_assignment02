# Project Name: DMoney API Testing

## Project Description:
This is a basic API testing project. Here Using API create 2 customer and 1 agent has been created and using admin API activated those users.From system account API depositted 5000 tk to agent account
 and then agent deposit 2000 tk to a customer.Then customer sends 1000 tk to another customer and another customer cashout 500 tk from the agent. Created both positive and negative test cases using API.

 ### Technology used:
 - NodeJS
 - Postman
 - Newman
 - HTML Report Extra

### How to Run:
- `npm init -y`
- `node report.js`
